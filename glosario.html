Glosario:

Glosario con algunos terminos usados en la ciberseguridad.

Ciberseguridad / Cybersecurity:
Es la práctica de proteger sistemas, redes y programas de ataques digitales.
Ciberataque / Cyberattack:
Es un intento malicioso de dañar, alterar, robar información o realizar actividades ilegales a través de medios digitales.
Malware / Malware:
Es un software malicioso diseñado para dañar o infiltrarse en un sistema informático sin el consentimiento del usuario.
Virus / Virus:
Es un tipo de malware que se propaga infectando archivos y programas, y puede causar daños en el sistema.
Gusano / Worm:
Es un tipo de malware que se propaga a través de redes informáticas y sistemas conectados, sin necesidad de intervención humana.
Troyano / Trojan:
Es un tipo de malware que se presenta como un programa legítimo pero que, en realidad, realiza acciones maliciosas sin el conocimiento del usuario.
Phishing / Phishing:
Es una técnica utilizada para engañar a los usuarios y obtener información confidencial, como contraseñas o datos bancarios, haciéndose pasar por una entidad de confianza.
Ingeniería Social / Social Engineering:
Es una técnica que manipula psicológicamente a las personas para obtener información confidencial o realizar acciones no autorizadas.
Hackeo / Hacking:
Es el acto de buscar y explotar debilidades en sistemas informáticos con el fin de obtener acceso no autorizado o realizar modificaciones no deseadas.
Ataque de Denegación de Servicio (DDoS) / Denial of Service (DDoS) Attack:
Es un intento de hacer que un servidor o red sea inaccesible para los usuarios legítimos, abrumándolo con una gran cantidad de solicitudes de conexión.
Intrusión / Intrusion:
Es el acceso no autorizado a sistemas informáticos o redes.
Firewall / Firewall:
Es una barrera de seguridad que controla y regula el tráfico de red entre una red interna y externa, ayudando a prevenir ataques no deseados.
Encriptación / Encryption:
Es el proceso de convertir información en un código seguro para protegerla de accesos no autorizados.
Autenticación / Authentication:
Es el proceso de verificar la identidad de un usuario antes de permitir el acceso a un sistema o recurso.
Contraseña / Password:
Es una secuencia de caracteres utilizada para autenticar la identidad de un usuario y acceder a sistemas o servicios.
Suplantación de Identidad / Identity Theft:
Es el acto de usar la información personal de alguien sin su consentimiento para cometer fraude u otros delitos.
Vulnerabilidad / Vulnerability:
Es una debilidad en un sistema o programa que puede ser explotada por un atacante para comprometer la seguridad.
Parche / Patch:
Es una actualización de software diseñada para corregir vulnerabilidades de seguridad o errores en un programa o sistema operativo.
Seguridad de la Red / Network Security:
Es el conjunto de medidas diseñadas para proteger la integridad, confidencialidad y disponibilidad de los datos en una red informática.
